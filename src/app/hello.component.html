Ejercicio:

Desarrollar una aplicación web usando el framework de Angular (5+) para capturar el domicilio de un cliente.

Parte 1:   El cliente deberá proporcionar su nombre y su código postal. Atravez de un REST API (url abajo en azul) la aplicación deberá obtener el estado, municipio y un listado de colonias que corresponden a dicho código postal.

https://api-codigos-postales.herokuapp.com/v2/codigo_postal/[codigopostal]

Parte 2:  El sistema mostrará en pantalla la información obtenida en la Parte 1 y solicitará al usuario la selección de una colonia. Asi como tmb pedirá Calle y Número. Cuando el usuario haya proporcionado todos estos datos, la aplicación aparecerá un botón con la leyenda Guardar.

Parte 3:  Al dar click al botón de Guardar, la información capturada se guardará en la sesion y la aplicación navegará a una nueva ruta en la cual se mostrará la información de la dirección capturada en la ruta anterior.

Parte 4:  Dar estilo a la aplicación implementando algun UI system. (Se podrá importar librerias de estilos como es bootstrap o material).
